## 🌟 Overview
Welcome to the **Supervised Machine Learning Bootcamp** repository! This project contains **22 distinct sessions**, each designed to introduce, implement, and master a specific Machine Learning concept. 

Every session is cleanly modularized into its own folder containing:
1. A strictly formatted **Jupyter Notebook (`.ipynb`)** with perfectly isolated code cells.
2. A dedicated **`README.md`** explaining the session's mathematical intuitions, business use cases, and algorithmic workflows.

---

## 🗂️ Table of Contents / Session Roadmap

### 📊 Phase 1: Data Mastery & Regression Basics
| Session | Topic | Key Concepts |
| :---: | :--- | :--- |
| **01 - 05** | **Data Engineering & Regression** | Pandas, Matplotlib EDA, Simple Linear Regression, Multiple Linear Regression, Polynomial Scaling, and Outlier Handling. |

### 🌳 Phase 2: Classification Algorithms
| Session | Topic | Key Concepts |
| :---: | :--- | :--- |
| **06 - 10** | **Classification Foundations** | Logistic Regression, K-Nearest Neighbors (KNN), Support Vector Machines (SVM), and Decision Trees. |
| **11 - 15** | **Ensembles & Boosting** | Random Forests, Bagging, AdaBoost, Gradient Boosting, and introduction to XGBoost. |

### 📈 Phase 3: Model Evaluation & Hyperparameter Tuning
| Session | Topic | Key Concepts |
| :---: | :--- | :--- |
| **16** | **Evaluation Metrics (Part 1)** | Confusion Matrix, True/False Positives, Precision, Recall, and F1-Scores. |
| **17** | **Evaluation Metrics (Part 2)** | ROC-AUC Curves, Log Loss, and evaluating heavily imbalanced datasets. |
| **18** | **Model Improvement (Part 1)** | K-Fold Cross-Validation, Stratified K-Fold, and Variance/Bias trade-offs. |
| **19** | **Model Improvement (Part 2)** | GridSearchCV, RandomizedSearchCV, and Class Balancing with **SMOTE**. |

### ⏳ Phase 4: Time Series & Forecasting
| Session | Topic | Key Concepts |
| :---: | :--- | :--- |
| **20** | **Time Series Analysis** | Trend/Seasonality Decomposition, Augmented Dickey-Fuller (ADF) Stationarity Tests, and Moving Averages. |
| **21** | **Time Series Forecasting** | Feature Engineering (Lag, Rolling, Temporal Extraction), ARIMA Modeling, Random Forest Forecasting, RMSE, and SMAPE. |

### 🏆 Phase 5: The Capstone
| Session | Topic | Key Concepts |
| :---: | :--- | :--- |
| **22** | **End-to-End ML Pipeline** | Capstone Case Study (Zomato Dataset). Handling missing text data, Encoding, Scaling, SMOTE hallucination, GridSearching Random Forests, and avoiding Data Leakage. |

## 📂 Datasets Utilized
This repository brings algorithms to life using a mix of real-world datasets and dynamic simulations:
* 🍔 **Zomato Restaurants Dataset** (Complex EDA & Imbalanced Classification)
* 🌡️ **Ahmedabad Daily Temperatures** (Time Series Decomposition & ARIMA)
* 💳 **Credit Card Fraud** (ROC-AUC & SMOTE evaluation)
* 🛒 **Flipkart Product Reviews / Sales** (Temporal Feature Extraction)
* 🌸 **Iris / Titanic** (Classic algorithmic benchmarking)

## 🚀 How to Run Locally

1. **Clone the Repository**
   ```bash
   git clone https://github.com/your-username/supervised-machine-learning.git
   cd supervised-machine-learning
   ```

2. **Install Dependencies**
   It is recommended to run these notebooks inside a virtual environment.
   ```bash
   pip install pandas numpy matplotlib scikit-learn imbalanced-learn statsmodels
   ```

3. **Launch Jupyter**
   ```bash
   jupyter notebook
   ```
   *Navigate to any `Session - X` folder to run the specific notebook!*
